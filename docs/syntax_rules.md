везде где это будет уместно ставить пробел после скобочек
пример как надо:
```c
void foo( int a, int b )
```
пример как НЕ надо:
```c
void foo(int a, int b)
```

называть константные переменные большими буквами
пример как надо:
```c
const int SIZE = 25;
```
пример как НЕ надо:
```c
const int size = 25;
```
